# Ebb and Ember — SOP: Pre-Tour Preparation
*2026-03-28 | Vessel type: floating sauna / vessel*

> **Purpose:** Ensure every tour launches safely, legally, and professionally. This SOP covers everything from the day before through launch.

---

## ⏰ Day-Before Checklist

- [ ] Confirm all bookings in FareHarbor — check headcount, any special requests
- [ ] Check weather forecast — wind, precipitation, wave height (if applicable)
- [ ] Send 48-hour reminder email to all confirmed guests (see Guest Comms SOP)
- [ ] Confirm staff/guide availability and assignments
- [ ] Review float plan / tour route for any new conditions
- [ ] Verify all gear is staged, charged, and accounted for

## 🌅 Morning-Of Checklist (Staff Arrival)

- [ ] Sauna stove/heater pre-heated to operating temp
- [ ] Towels, water, cooling buckets stocked
- [ ] PFDs accessible if vessel is moored on water
- [ ] Safety briefing card posted inside sauna
- [ ] First aid kit stocked
- [ ] Entry/exit ladder clear and secure
- [ ] Emergency contact number posted inside

## 👋 Guest Check-In Procedure

- [ ] Greet guests by name when possible
- [ ] Verify headcount against booking manifest
- [ ] Collect signed liability waiver (digital preferred — DocuSeal/FareHarbor)
- [ ] Fit PFDs — guide checks every guest individually
- [ ] Safety briefing delivered (see script below)
- [ ] Confirm any medical/physical limitations
- [ ] Photo opportunity at launch (social media + guest memory)

## ☁️ Weather Hold Protocol

**Decision maker:** Guide lead / Manager on duty

| Condition | Action |
|---|---|
| Severe weather advisory | Cancel, full refund |
| Lightning | Cancel / shelter |

**Weather hold communication:** Use guest communication template in Guest Comms SOP

## 📝 Post-Tour Checklist

- [ ] All gear returned, rinsed, and stored
- [ ] Incident report filed if any injury/incident occurred
- [ ] Guest count confirmed out
- [ ] Review request triggered in GHL (auto if connected, else manual SMS)
- [ ] Any feedback noted for manager coaching log
